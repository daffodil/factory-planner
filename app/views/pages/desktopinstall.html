Install the TRT desktop

Overview

Getting up and running on a desktop requires the following steps:

  1. Install Python - this interperator runs the source code
  2. Install PyQt - makes the windows and widgets
  3. Install Git - for downloading and updating the source code
  4. TRT Code Checkout - download the trt application code
  5. Run and Login - launch the TRT application

1: Install Python3

  • Required is the latest in the Python 3+series
  • Download and run the Windows MSI Installer
  • Accept all the defaults. A new directory c:\python34 will be created
  • https://www.python.org/download/

2: Install PyQt5

  • Required is the PyQt5 - Binary package
  • Download and run the Windows Installer
  • Accept all the defaults. fyi will install into c:\python34
  • http://www.riverbankcomputing.com/software/pyqt/download5

3: Install Git

  • Required is the latest msysgit - git for windows
  • Download and run the Windows MSI Installer
  • On the "Adjusting PATH environment", select "Use Git from Windows Command Prompt", otherwise accept defaults. fyi will install into c:\ProgramFiles\*
  • http://msysgit.github.io/

4: TRT Code Checkout

  • This step downloads/clones the source code for the trt desktop using git
  • The source code is hosted by bitbucket.org
  • Start Git Gui with >> Start > All Programs > Git > Git GUI
  • Select "Clone Existing repository"
  • Source Location =
  • Target Directory =
  • Click Clone button
  • On completion you will see a "multipane window", just dismiss and close.

5: Run and Login

  • Navigate to the c:\zgit\trt-desktop directory
  • To create a shortcut, click on the "TRT Desktop" link, and drag onto the Windows Start Button
  • Start the TRT Desktop by double clicking
  • After a few moments, you should see a login screen
  • This means that the install works, so login to play site and enjoy
